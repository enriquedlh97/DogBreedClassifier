# DogBreedClassifier
_Model used for the final project of TI2022 - Dog Breed classifier app for local Veterinary_

## Getting started 🚀

To get started make sure you set up the anaconda environment from the .yml file.

```bash
conda env create -f environment.yml
```

Make sure you activate the environment. 
```bash
conda activate myenv
```

And verify that it was properly installed.
```bash
conda env list
```
